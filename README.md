##### 使用说明
1. 首先要有python环境啦
2. pip install requests -i https://mirrors.tuna.tsinghua.edu.cn/
3. pip install urllib3 -i https://mirrors.tuna.tsinghua.edu.cn/
4. random json time好像不需要install
5. 打开cmd窗口
6. 职教云：输入python zjy.py 即可
7. Mooc：输入 python mooc.py 即可
